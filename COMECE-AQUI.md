## 🎯 Primeiro Passo: Abra o Site

### Opção 1: Abrir arquivo HTML diretamente (Mais simples)

1. Abra a pasta `RETIRO-SITE` no seu computador
2. Clique 2 vezes em `index.html`
3. O site abre no navegador

**Pronto! Seu site está funcionando!** ✨

---

### Opção 2: Usar um servidor local (Melhor para desenvolvimento)

#### No macOS (Python):
```bash
# Abra o Terminal
cd ~/Documents/RETIRO-SITE
python3 -m http.server 8000

# Depois acesse: http://localhost:8000
```

#### Com http-server (qualquer OS):
```bash
# Instale (primeira vez):
npm install -g http-server

# Execute:
cd ~/Documents/RETIRO-SITE
http-server

# Acesse: http://localhost:8080
```

## 🎨 Próximos Passos Recomendados

### 1️⃣ **Adicione Imagens** (Torna o site mais atrativo)
- Coloque imagens em `assets/images/`
- Edite `index.html` para referenciar as imagens

### 2️⃣ **Customize Cores e Textos**
- Abra `index.html` em um editor de texto
- Procure por textos para editar
- Procure por classes Tailwind para mudar cores

### 3️⃣ **Teste em Mobile**
- Redimensione a janela do navegador
- Abra DevTools (F12) → Toggle device toolbar
- Veja como fica em celulares

### 4️⃣ **Configure Email e Formulário**
- Veja `BACKEND-GUIDE.md` para integrar um backend
- Assim seus formulários vão processar de verdade

### 5️⃣ **Deploy (Coloque Online)**
- Hospede em Netlify, Vercel ou GitHub Pages (grátis)
- Veja instruções em `README.md`

## 🆘 Troubleshooting

**P: O site não abre?**  
R: Certifique-se que abriu o arquivo `index.html` no navegador

**P: Estilos não aparecem?**  
R: Limpe o cache (Ctrl+Shift+Delete) e recarregue a página

**P: Quer testar no celular?**  
R: Se estiver em rede local, use: `http://seu-ip:8000`

**P: Como mudar cores?**  
R: Veja `CUSTOMIZACAO.md`
